# How to Install

Prerequisites:
- Node.js
- MongoDB

Install the necessary dependencies
```
npm install
```

We use nodemon to run the project:
```
npm install nodemon -g
```

To run the project:
```
nodemon app.js
```
Then go to the browser: [http://localhost:3000/campgrounds](http://localhost:3000/campgrounds)
